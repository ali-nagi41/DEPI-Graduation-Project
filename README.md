## Tech Stack
- Frontend: React, Vite, TailwindCSS
- Backend: Node.js, Express, Socket.io
- Database: MongoDB
- AI Integration: Google Gemini API

## Setup Instructions
1. Install dependencies: `cd server && npm ci && cd ../client && npm ci`
2. Start the server & client: `npm start`
